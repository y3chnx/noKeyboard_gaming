## ℹ️ How to Use?

- Download my repository on your computer.
```bash
git clone https://github.com/y3chnx/noKeyboard_gaming.git
```

- Install all the things that you need.
```bash
pip install requirements.txt
```

- Open up the Game
```bash
python3 main.py
```


## ℹ️ How to play?

You are playing against a simple robot that runs with random.random() function
You can have your thumbs up to make your plate go up, and make a fist to make your plate go down
